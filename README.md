# Course website for Introduction to Jugger

live now at
[jugger.studentorg.berkeley.edu/decal-fa26/](https://jugger.studentorg.berkeley.edu/decal-fa26/)!

This repo is the source for the course website for the Berkeley DeCal
(student-led course) *Introduction to Jugger*, offered as ERG 98 in
the Fall 2026 semester.

The goal is to create a repo with all of the content essentially ready
to go so that it can be forked by anyone trying to teach a similar
class. This is all still a work in progress, as the course has not
been taught yet, so things may change a lot! But we hope that by the
end of the semester, we'll have a genuinely useful resource for people
to use.
